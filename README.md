## API implementation using Nodejs

#### features:
1. Users can register.
2. Users can login.
3. Users can add an article.
4. Users can view all articles, sort and pagination.



### How to use:-
  1. first clone the repo
  2. ### `npm install` 
  3. ### `npm start`

there are four APIs
